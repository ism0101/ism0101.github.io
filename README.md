# Irene Sanchez Martin Academic Website

This is a static website designed to be published with GitHub Pages.

## Files

- `index.html`: homepage structure
- `styles.css`: site styling
- `script.js`: small footer year enhancement
- `.nojekyll`: tells GitHub Pages to serve the site as-is

## Publish On GitHub Pages

1. Create a GitHub repository.
2. Push these files to the repository.
3. Open the repository on GitHub.
4. Go to `Settings`.
5. Open `Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select your main branch and `/ (root)`.
8. Save and wait for GitHub Pages to publish the site.

If the repository is named `<username>.github.io`, this site will be served as your main user website.

If the repository has another name, the site will be served under a project URL like:

`https://<username>.github.io/<repository-name>/`

## Customization Notes

- Replace the headshot placeholder in `index.html` with a real image when ready.
- Add your bio, email, CV, and profile links.
- Replace the publication placeholders with real entries.
